# ISTQB CTFL v4.0 — Study & Practice Site

A local/hosted study site for the **ISTQB Certified Tester Foundation Level (CTFL) v4.0** exam, tailored for candidates in India.

🌐 **Live Site:** [https://maniziva.github.io/istqb-ctfl](https://maniziva.github.io/istqb-ctfl)

---

## Features

- **Full Syllabus** — All 6 chapters with content, key terms, and learning objectives
- **110+ Practice Questions** — Official Sample Exam A questions + India exam-style questions
- **Quiz Mode** — Filter by chapter, set question count, practice or exam simulation mode
- **Full Exam Simulation** — 40 questions with 60-minute countdown timer (65% to pass)
- **Flashcards** — 20 key concept cards covering principles, formulas, and review types
- **Quick Reference** — 7 principles, testing quadrants, review types table, test levels
- **Progress Tracking** — Per-chapter score saved in browser localStorage
- **Search** — Search across all questions and syllabus content

## Usage

### Run Locally
Just open `index.html` in any browser — no server or build step needed.

```bash
open index.html
```

### Host on GitHub Pages
1. Push to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **Branch: main / folder: / (root)**
4. Site will be available at `https://<username>.github.io/<repo-name>`

---

## Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no dependencies
- Progress saved in browser `localStorage`
- Works fully offline after first load
